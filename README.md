# ExamPro

ExamPro is a secure and modern desktop-based online examination system developed using Java and JavaFX.  
The application is designed to simulate real-world computer-based testing environments with secure authentication, fullscreen examination mode, timed assessments, and administrative controls.

---

## Features

- Secure role-based authentication
- Student and Administrator login system
- Fullscreen examination environment
- Timer-based examination engine
- Multiple Choice Questions (MCQs) support
- Subjective question support
- Real-time result processing
- Question management system
- User registration and login
- Clean and modern JavaFX user interface
- Database integration for storing users, exams, and results

---

## Technologies Used

- Java
- JavaFX
- Maven
- JDBC

## Installation and Setup

### Prerequisites

Make sure the following software is installed on your system:

- Java JDK 17 or above
- Maven
- MySQL Server
- IntelliJ IDEA or any compatible IDE

---

### Clone the Repository

```bash
git clone https://github.com/your-username/exampro.git
```

---

### Database Setup

1. Create a MySQL database.
2. Import the provided SQL file into MySQL.
3. Update the database configuration inside the project files.

---

### Run the Application

Using Maven:

```bash
mvn clean javafx:run
```

Or run the main application class directly from the IDE.
